# Team 3 Taxi Project

This repository contains the completed work for an end‑to‑end Data & AI demonstrator project. The project delivers a simplified but coherent Lakehouse solution that includes data ingestion, data refinement, analytics outputs, a machine learning experiment tracked with MLflow, and a retrieval‑augmented generation (RAG) workflow.

## Project Summary
- Ingestion (Bronze) was implemented to bring raw data into the Lakehouse.
- Refinement (Silver) transformed and cleaned the data for analysis.
- Analytics and dashboarding (Gold) outputs were produced for stakeholder insights.
- A machine learning experiment was completed using MLflow, creating a fare predictor called taxSEE.
- A functional RAG component was created to demonstrate retrieval‑based question answering.

## Dataset
The project used the NYC Yellow Taxi Trip dataset:
https://www.kaggle.com/datasets/elemento/nyc-yellow-taxi-trip-data/data
